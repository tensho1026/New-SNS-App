## 📄 概要
SNS風の投稿アプリ。ユーザーは画像付きの投稿プロフィール編集、コメント、いいねが可能。

## 🔧 使用技術
- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase（DB,, ストレージ）
- Clerk（ログイン認証）
- Prisma（ORM）
- Zustand（状態管理）
- React Hook Form + Zod（バリデーション）
- Hono(api作成)
